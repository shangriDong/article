Git 三个工作区域关系


**本地库关联远程库，在本地仓库目录运行命令：**
```
 $git remote add origin git@github.com:nanfei9330/learngit.git
```

**提示出错信息：fatal: remote origin already exists.**
```
$ git remote rm origin
```

之后再add origin
